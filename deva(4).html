WHAT IS COMMENT TAGS:

Comment tags are used in programming and markup languages (like HTML) to include notes or explanations in the code that are ignored by the computer when the code is run or displayed. They're meant for humans—developers or anyone reading the code—to understand it better.

USES OF COMMENT TAG:

  1. To explain what the code does
  2. To leave TODO notes or reminders
  3. To temporarily disable parts of the code
  4. To help others (or your future self) understand your logic
